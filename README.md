# react-firebase-todo-workshop
This is initially set up with "npm create vite@latest (app_name) --template react

Vite is a JavaScript bundler that helps develop a React app more efficiently. It bundles React components and dependencies into files for production, and you can also see UI updates instantaneously. 

Clone this repository, and after cloning, run **npm install** to install project dependencies. 

Run **npm run dev** to make sure the development server starts and work.
